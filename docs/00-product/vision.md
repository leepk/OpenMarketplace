## Long-term Vision

OpenMarketplace is not only a classifieds app.

It is a marketplace engine that can power multiple marketplace products through shared core domains.

```txt
OpenMarketplace Engine
├── Classifieds
├── Vehicle Marketplace
├── Real Estate
├── Service Marketplace
├── Job Board
├── Pet Marketplace
└── Business Directory
```

## Core Vision

- One engine
- Multiple marketplace products
- Clean domain boundaries
- Configurable commerce/ads/promotions
- Provider-based integrations
- Production-ready V1
- Public/community friendly documentation

## V1 Scope

The first public product is Classifieds with:

- Listings
- Categories
- Media uploads
- Feed/list view
- Map view basic
- Search filters
- Messaging
- Notifications
- Likes/comments/favorites
- Seller trust badges/reviews
- Packages/promotions
- Stripe payment provider
- Invoice PDF
- Ads placement
- Moderation basic
- Admin operations
