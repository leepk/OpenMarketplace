## V1 Scope

### Customer

- Register/login
- Profile and avatar
- Browse home feed
- Search listings
- View map
- View listing detail
- Post listing
- Upload listing images
- Choose package
- Checkout for paid package
- Message seller
- Receive notifications
- Save/favorite listing
- Like listing
- Comment listing
- Report listing/user/message/comment
- View billing history
- Download invoice PDF
- View seller profile, badges, rating

### Seller

- Create and manage listings
- Pause/resume/mark sold
- Promote listing
- Receive messages
- View basic listing stats
- See moderation status
- Receive listing approved/rejected notifications

### Admin

- Dashboard
- Listing review
- User management
- Moderation queue
- Reports
- Messages monitor basic
- Comments/reviews moderation
- Trust/badges management
- Packages
- Orders/payments/invoices
- Ads/promotions
- CMS legal pages
- Email templates
- Settings
- Audit logs
- Health checks

### Platform

- Modular monolith
- PostgreSQL
- Media service
- Storage abstraction
- Payment provider abstraction
- Email provider abstraction
- Feed engine
- Background jobs
- Docker deployment

## Not in V1

- Native mobile app
- Microservices
- Payout/settlement
- Full AI moderation
- Full booking calendar
- Advanced recommendation
- Multi-tenant billing
- Complex tax provider
