## V1 Moderation Policy

Moderated content:

- Listings
- Images
- Messages
- Comments
- Reviews
- Ads
- User profile text

Checks:

- Blocked keywords
- Suspicious URL
- External payment keywords
- Spam repeated text
- Duplicate title/description
- Duplicate image checksum
- File type/size validation
- Report history
- User trust score

Risk score:

```txt
0-30    Auto approve
31-70   Human review
71-100  Reject or urgent review
```
