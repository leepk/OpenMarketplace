## Scope Format

Each document separates:

```txt
Long-term Vision
V1 Scope
Future Scope
Not in V1
```


## Long-term Vision

Runtime configuration and feature flags.

## V1 Scope

Upload limits, moderation thresholds, currency/tax, listing expiration, package/ad config.

## Future Scope

Tenant settings, rollout flags.

## Not in V1

Advanced/enterprise features remain future unless explicitly listed in V1.
