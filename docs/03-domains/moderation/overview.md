# Moderation Overview

## Status

Reviewed Draft

## Scope Format

Each document separates:

```txt
Long-term Vision
V1 Scope
Future Scope
Not in V1
```


## Long-term Vision

Protects users from spam, scam, unsafe content.

## V1 Scope

Rule engine, reports, duplicate checks, review queue, risk score.

## Future Scope

AI/OCR/NSFW, appeals, advanced ML.

## Not in V1

Advanced/enterprise features remain future unless explicitly listed in V1.
