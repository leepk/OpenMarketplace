# Feed Engine Overview

## Status

Reviewed Draft

## Scope Format

Each document separates:

```txt
Long-term Vision
V1 Scope
Future Scope
Not in V1
```


## Long-term Vision

Home/search/category/profile pages consume FeedItem[] instead of raw listings.

## V1 Scope

Listings, featured listings, inline ads, simple ranking, duplicate removal.

## Future Scope

Recommendations, personalization, trending, A/B ranking.

## Not in V1

Advanced/enterprise features remain future unless explicitly listed in V1.
