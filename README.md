# OpenMarketplace

OpenMarketplace is a modern marketplace platform built with:

* .NET 9
* ASP.NET Core
* PostgreSQL
* React
* Next.js
* Docker

## Features

* Marketplace listings
* Categories & Subcategories
* Maps
* Banner Ads
* Premium Listings
* User Management
* Admin Portal
* Multi-language
* Responsive UI

## Quick Start

```bash
git clone https://github.com/your-account/OpenMarketplace.git
```

Follow the setup instructions in the `/docs` folder.
