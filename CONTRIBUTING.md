# Contributing

Thank you for your interest in OpenMarketplace.

We welcome:

* Bug reports
* Documentation improvements
* Pull requests
* Feature suggestions

Please ensure that:

* Code follows the existing style.
* New features include documentation.
* Pull requests are focused and easy to review.

By submitting a contribution, you grant the project owner the right to include your contribution in future versions of OpenMarketplace under any license chosen by the project owner.
